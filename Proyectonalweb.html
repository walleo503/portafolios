Periodo 1.

Breves resúmenes.

Gestión de servidores web

Ente apartado se explica que son los servidores web ,los tipos de servidores que existen, junto con sus características y su función.

Virtualización.

La virtualización es la creación dé una representación  basada en software o virtual de una entidad física, como ejemplo tenemos aplicaciones, servidores y  almacenamiento virtual.

Contenedores.

Los contenedores son una tecnología para empaquetar y ejecutar aplicaciones en  diferentes sistemas operativos tales como Windows y  Linux, esto atreves de  diversos entornos locales y en la  nube.

Periodo 2.

Breves resúmenes.

Contenedores Linux o LXC

 es un conjunto de uno o más procesos separados del resto del sistema. Todos los archivos que se necesitan para ejecutarlos provienen de una imagen diferente, lo cual significa que los contenedores de Linux son portátiles y uniformes durante todas las etapas: desarrollo, prueba y producción.

Hyper -V

es una tecnología de virtualización basada en hipervisor para ciertas versiones x64 de Windows. El hipervisor es el núcleo de la virtualización. Es la plataforma de virtualización específica del procesador que permite que múltiples sistemas operativos aislados compartan una sola plataforma de hardware.

Servlets Java.

son componentes de software que se ejecutan en un servidor web y permiten crear aplicaciones web dinámicas.

Periodo 2.

Breves resúmenes.

Servidor web

Es un software o hardware que se encarga de gestionar y servir contenido web a través de Internet. Es fundamental en el proceso de almacenar, procesar y entregar páginas web, aplicaciones y otros recursos a los usuarios finales.

Servidor DNS.

Es un software de servidor especial que utiliza una base de datos DNS para responder a las consultas relativas al DNS. Dado que los servidores DNS suelen estar alojados en hosts dedicados, los ordenadores que contienen los programas correspondientes también se denominan servidores DNS.

Periodo 3.

Breves resúmenes.

Cloud computing

es la disponibilidad bajo demanda de recursos de computación como servicios a través de Internet. Esta tecnología evita que las empresas tengan que encargarse de aprovisionar, configurar o gestionar los recursos y permite que paguen únicamente por los que usen.

Load Balancer Página.

 es una técnica utilizada para el reparto del trabajo entre diferentes computadores, generalmente entre los nodos de un cluster. Suele ser un dispositivo o computador dedicado, provisto de un software que gestiona el trabajo del cluster evitando los cuellos de botella y recibe el nombre de Balanceador de Carga.
